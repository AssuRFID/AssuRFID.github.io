AssuRFID: security for the modern world

A new entry into the world of security, AssuRFID brings ease and simplicity to the ever more important task of keeping our workplaces, buildings and technology secure, all while guaranteeing best-in-class security.

Technology from the Future

AssuRFID combines wireless radio technology with the latest in Electrocardiography, enabling users to make use of the most secure, affordable authentication system.

It uses uncloneable, anti-counterfeited Radio Frequency Identification (RFID) tags embedded into a wristband worn by the user, along with their cardiac rhythm, in order to enable their security with 2-factor authentication.

To ensure easy administration of the system, AssuRFID employs a custom server backed by a database, to which the sensors across the whole campus connect.

AssuRFID is versatile

When designing AssuRFID, we realised that our technology could be applied to numerous settings. These range from banks, prisons and workplaces, to maternity wards, schools and business-places.

These applications each share a need for easy to use yet secure authentication to open doors, sign into systems, log onto computers and much, much more.

In addition we will have custom models for maternity wards, with tags on the mother and embedded in the baby grow, so the they can only leave with the right child.

Check out our prototype

Our working prototype, named Assured, comprises a suite of custom applications which implement AssuRFID's vision, including a web server which clients connect to, and a custom Graphical User Interface (GUI) to add new users and log them in.

Our code is fully open source, and can be found online in our GitHub repositories for assured-server and assured-gui.
